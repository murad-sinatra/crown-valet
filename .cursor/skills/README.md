# Project Skills

This folder contains project-scoped Cursor skills for Crown Valet. Agents should read these skills when working on product planning, design, implementation, or documentation so future changes stay aligned with the existing product direction.

## Available Skills

- `crown-valet-product`: Use for Crown Valet product development, MVP planning, design system decisions, and feature implementation guidance.

## Maintenance

- Keep skills concise and point to source documents instead of duplicating long documentation.
- Update skills when the product scope, architecture, design system, or MVP plan changes.
- Prefer project skills here over personal skills when the guidance should travel with the repository.

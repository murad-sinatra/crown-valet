# Crown Valet

A simple Nuxt landing page for a valet parking app.

## Setup

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

This project uses Nuxt 3. If install or build fails with an engine warning, switch to Node.js 20.19 or newer before running the commands above.
